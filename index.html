Conga.js - a Node.js framework.

Conga.js allows you to quickly build and maintain organized and flexible web applications.

Highlights

Configuration

Hierarchical environment configuration using YML

Routing

Simple and flexible annotation based routing.

ODM

Bass.js, the annotation based ODM for NoSQL

Bundles

Build your application from existing or custom re-usable bundles

Other cool stuff

  • Built on top of Express 3
  • Integrated websockets through Socket.IO
  • Simple REST service generation
  • Integrated Bower
  • Dependency Injection
  • Automatic asset compilation
  • Share server-side code and templates with browser

Install Now!!!

Install the global conga command:

$ npm install -g conga

Example Controller

/**
 * @Route("/")
 */
function DefaultController(){};

DefaultController.prototype = {

	/**
	 * @Route("/", name="default.index", methods=["GET"])
	 * @Template
	 */
	index: function(req, res){
		res.return({ message : "Hello World!!!" });
	},

	/**
	 * @Route("/hello/{name}")
	 */
	hello: function(req, res){
		res.return({ message : "Hello " + req.param("name") });
	}
};

module.exports = DefaultController;

Example Model

/**
 * @Bass:Document(name="User", collection="users")
 */
function User(){};

User.prototype = {

	/**
	 * @Bass:Id
	 * @Bass:Field(type="ObjectID", name="_id")
	 */
	id: null,

	/**
	 * @Bass:Field(type="string")
	 */
	email: null,

	/**
	 * @Bass:Version
	 * @Bass:Field(type="Number")
	 */
	version: 0,

	/**
	 * @Bass:CreatedAt
	 * @Bass:Field(type="Date")
	 */
	createdAt: null,

	/**
	 * @Bass:UpdatedAt
	 * @Bass:Field(type="Date")
	 */
	updatedAt: null

};

module.exports = User;
