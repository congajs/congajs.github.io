Quick Start

Install

In order to get started, you first need to install the conga module:

$ npm install -g conga

This will now give you the "conga" command line tool.

Create a project

$ conga create:project myproject

This will create a directory named "myproject" in your current directory.

Launch the application

$ cd myproject
$ conga play

This will launch the default sample application at http://localhost:3000

Start learning!!!

Congratulations! You're up and running. Now learn how to do stuff!
